How to use Canvas

Create blocks

  • Double-click on empty canvas to add a text block.
  • Ctrl + drag on empty canvas to draw a process block at the size you want.
  • Ctrl + click drops a default process block.
  • Keyboard: Ctrl + P drops a process block at the cursor.

Edit a block

  • Double-click the title or description to edit it in place (Enter adds a new line, click away to finish).
  • Right-click a block to pick a colour (swatches, a custom colour, or Default).
  • Select a block and drag any side or corner to resize it.
  • Add links in the description: a line like L - Label - https://… becomes a clickable link, and bare URLs auto-link. Links stay clickable in the exported SVG.

Connect blocks

  • Select a block and click its → button, then click the target block. Hold Space and drag to pan to an off-screen block, then click it.
  • Or drag from one of a block's edge dots onto another block's dot.
  • Select a connection to get endpoint dots (drag to re-anchor) and a hollow midpoint dot (drag out to add a bend point). Double-click a bend point to remove it.
  • Double-click a connection to add a multi-line label.

Select & move

  • Alt + drag (or Shift + drag) on empty canvas to box-select blocks and groups.
  • Alt/Shift + click a block toggles it in the selection.
  • Drag to move; Delete removes the selection. Ctrl+Z / Ctrl+Y undo / redo.
  • Pan: drag empty canvas or hold Space and drag. Zoom: mouse wheel.

Groups

  • Ctrl + Shift + drag to draw a dotted group box around blocks.
  • Click the group's label or its border line to select it; double-click the label to rename.
  • Drag the label to move the group with its blocks; drag a side/corner to resize.

Save & share

  • Set the file name in the top-left field — it's used by Save PNG, Save SVG, and Save Diagram.
  • Save Diagram writes an editable .json; Open Diagram loads it back.
  • Full Screen shows just the canvas (Esc to exit).
